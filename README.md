# Brain Tumor Detection using Deep Learning

## Overview

This project detects brain tumors from MRI images using a deep learning model (ResNet18).

## Features

* Binary classification (Tumor / No Tumor)
* Evaluation using accuracy, precision, recall
* Confusion matrix analysis
* Streamlit web app for predictions

## How to Run

```bash
python -m streamlit run app.py
```

## Dataset

Brain MRI images (manually split into train and test)

## Note

This model is for educational purposes only and not for medical use.
